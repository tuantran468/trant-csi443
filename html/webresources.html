Web Resources

  • W3Schools
    The W3Schools site contains a comprehensive reference of many web resources. W3Schools is a website for learning about HTML, CSS, JavaScript, JSON, PHP, Python, AngularJS, SQL, Bootstrap, Node.js, jQuery, XQuery, AJAX, and XML. It was created in 1998. Its webpages have thousands of code examples and a editor that lets the user try the code and excute it to see what it would look like.

    *Rating 5/5

  • GetBootstrap
    The GetBootstrap site contains documentations, examples of the bootstrap reference. It also has a link to the premade bootstrap themes. It gives you examples of the layout, framework and some custom component. Bootstrap was original release in August of 2011 and have now been updated to version 4.

    *Rating 5/5

  • jQuery
    The jQuery site is a learning site. It has sections you can read about to learn jQuery. It's goal is to provide user with references to all aspects of using jQuery. It has a contributing section where it shows user how to contribute to the site to help other developers because learning how and when to use jQuery is a different process for each and every web developer, depending largely on experience with the primary tools for front-end development.

    *Rating 5/5

  • CSS-Trick
    CSS-Tricks is really about building websites and all that entails, mostly from a front-end perspective. It has an article/blog section where users can post an article and others can comment relating to CSS, HTML, and JavaScript. It has videos to teach you about front end development.

    *Rating 5/5

  • MDN Web Docs
    The MDN web docs site contains a comprehensive reference of many web resources including html. This links talks about meta element. Metadata is data that describes data, and HTML has an "official" way of adding metadata to a document. Many meta element contains a name and content attribute which specifies the name of the meta and describes the meta content.

    *Rating 5/5

  • TutorialsPoint
    The TutorialsPoint site is a site created for those tht prefer to learn new skills at their own pace and in the comfort of their own home. Their missing is to deliver simple and easy learning. They has a large file sharing server so users can share files and they have library tutorial videos to learn about your subject.

    *Rating 5/5

  • Lynda
    The Lynda site is an American massive open online course website offering video courses taught by industry experts in software, creative, and business skills. It was founded in 1995 by Lynda Weinman as lynda.com before being acquired by LinkedIn in 2015. In 2016 LinkedIn was acquired by Microsoft. Lynda is a paid site, but it is free if you have a public library card. If you want to get access you should use the link from the public library site and enter your library card.

    *Rating 5/5
